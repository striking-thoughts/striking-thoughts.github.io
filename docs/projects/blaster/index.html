Rocket Jump

Rocket jump around the map in a capture the flag online shooter!
You can explore the game on your own, but it is more fun to play with some friends.

What is Rocket Jump?

I had some free time after work, so I decided to learn how to use the Unreal Engine.
I explore it a bit on my own and really enjoyed it. So I found a course that could serve as a hands on guided tour around some of the features of the engine.

After completing it, I took away a few things we had done and added a few of my own.

Can I play it this?

You should be able to launch and play the game solo, but it is not much of a game with just one person. I never intended to release this to a wide audience, so playing it with friends might require some additional steps on your part to change the steam app id to point to your own app id. I don’t recommend using the 480 steam app id.

I do have a steamappid that I use for my own testing, if you want, I can add you and others to the group so you can test it out. if you have any questions, feel free to contact me.
